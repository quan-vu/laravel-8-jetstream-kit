# Laravel 8 Starter

Step 1: Install libraries with composer

```shell
composer install
```

Step 2: Install Vue

```shell
yarn install && yarn dev
```

Step 3: Run migration database

```shell
php artisan migrate
# php artisan migrate:refresh --seed
php artisan storage:link
php artisan serve
```
